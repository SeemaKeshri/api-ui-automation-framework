| Package     | Purpose                                      |
| ----------- | -------------------------------------------- |
| `base`      | Base API configuration (BaseTest, BaseAPI)   |
| `client`    | API request methods (GET, POST, PUT, DELETE) |
| `constants` | Base URLs, endpoints, constants              |
| `models`    | POJO classes (User, CreateUserRequest, etc.) |
| `utils`     | Config reader, JSON utilities, random data   |
| `reports`   | Extent Report setup                          |
| `api`       | Actual TestNG test classes                   |
| `schemas`   | JSON schema files for validation             |

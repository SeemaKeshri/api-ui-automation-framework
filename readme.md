# API UI Automation Framework

Enterprise-style automation framework built using:

- Java
- REST Assured
- TestNG
- Maven
- Jackson
- Extent Reports

This framework demonstrates API automation using reusable components and follows best practices for maintainability and scalability.

## Configuration

Copy:

config.properties.example

to

config.properties

Update:

api.key=<your_api_key>
